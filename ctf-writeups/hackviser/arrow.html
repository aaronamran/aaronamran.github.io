This writeup focuses on practicing basics of the Telnet service. Telnet (Telecommunication Network) is a protocol used for accessing remote computers over the internet. It provides a text-based interface for users to communicate with and control another machine, but lacks encryption, making it less secure.

Tasks

Question 1: Which port(s) are open?

Run a Nmap scan on the target IP using nmap -sV -sC -A [target IP].

Answer: 23

Question 2: What is the running service name?

Answer: Telnet

Question 3: What is the hostname?

For this question, we can simply use telnet to connect to the target machine and observe the hostname.

Answer: arrow

Question 4: What's the username:password you use to connect to telnet?

To find the username and password, we can observe the hint given when using telnet to connect to the target machine.

Answer: root:root

Question 5: What is the working directory location when you connect to telnet?

After successfully connecting to the target machine, use the command pwd to check the current working directory.

Answer: /root
